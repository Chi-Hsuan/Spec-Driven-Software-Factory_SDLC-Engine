# Wireframe Schema

---

# 1. 必要結構

## 1.1 Page 必須存在
- 至少 1 個 Page

---

## 1.2 每個 Page 必須包含

- Page 名稱
- Page 目的
- Layout 結構
- 區塊拆解

---

## 1.3 Layout 規則

必須包含：

- Header / Body / Sidebar（或等價結構）
- 區塊不能只有文字描述，必須有結構

---

## 1.4 區塊（Section）

每個 Section 必須包含：

- 名稱
- 位置
- 目的

---

# 2. 驗證規則

❌ 無 layout → invalid  
❌ 無 page → invalid
